# Pizza Sales Analysis Dashboard  (SQL & Power BI)

# Project Overview
This project analyzes pizza sales data using SQL for data extraction and Power BI for visualization.  
It identifies key sales trends, top-performing pizzas, and customer purchase behaviors.

# Tools & Skills Used
- SQL (Data Extraction, Aggregation, KPI Calculation)
- Power BI (Dashboarding, DAX, Data Modeling)
- Data Analysis & Visualization
- Business Insights Reporting

# Key Features
- Total Revenue, Orders, and Average Order Value KPIs
- Daily, Monthly, and Category-wise Sales Trends
- Top 5 & Bottom 5 Pizzas by Revenue and Quantity
- Interactive Power BI dashboard with slicers and filters
